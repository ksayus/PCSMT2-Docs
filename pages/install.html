    # PCSMT2 安装文档

    # 第一步:下载

    从 GitHub 下载最新版本的 PCSMT2 代码 PCSMT2 (opens new window) 或者从网站下载 PCSMT2 (opens new window)

    # 第二步:安装

    下载后将文件移入一个空文件夹中，文件夹名称可以自定义。

    # 第三步:运行

    随后启动 PCSMT2.exe 文件 程序会自动检测Java的安装,会自动补全缺失版本 程序会创建多个文件夹和文件 此时目录下结构为

    文件 / 文件夹 说明
    server 服务器文件夹
    logs 日志文件夹
    resource 资源文件夹
    server_storage_size 服务器存储大小
    config.json 配置文件
    java_version.json java路径
    latest.txt 最新一次启动的服务器
    PCSMT2-v版本号.exe 程序文件

    稍作等待,程序会完成初始化,并启动图形界面

    至此,PCSMT2 安装完成
