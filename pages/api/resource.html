# 服务器资源

  • /api/server/info/{ServerName}/excel 这个接口用于获取指定服务器的Excel表格.

TIP

这个接口返回的是一个Excel文件,文件内容包含:

  • 服务器名称
  • 服务器版本
  • 核心类型
  • 上次启动时间
  • 启动次数
  • 存储占用

请求参数:

  • ServerName: 服务器名称 -> str

使用方法:

GET /api/server/info/{ServerName}/excel

响应数据:

二进制流(文件)

文件格式:

server1.xlsx

服务器名称 服务器版本 核心类型 上次启动时间 启动次数 存储占用
server1 1.21.1 fabric 2023/08/02/13:00:00 11 12.32GB
  • /api/server/list/excel 这个接口用于获取所有服务器的Excel表格.

TIP

这个接口返回的是一个Excel文件,文件内容包含:

  • 服务器名称
  • 服务器版本
  • 核心类型
  • 上次启动时间
  • 启动次数
  • 存储占用
  • 底部的sheet工作表,用于存储不同的服务器信息
    • 一个服务器一个sheet工作表

使用方法:

GET /api/server/list/excel

响应数据:

二进制流(文件)

文件格式:

serverList.xlsx

服务器名称 服务器版本 核心类型 上次启动时间 启动次数 存储占用
server1 1.21.1 fabric 2023/08/02/13:00:00 11 12.32GB
sheet1 sheet2
server1 server2
