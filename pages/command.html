# 命令行使用

所有命令都需要在 PCSMT2.exe 所在目录下运行。 获取所有命令的列表

help

获取单个命令的帮助信息,可以使用 help 命令名 例如

help start_server

# 服务器类

命令 说明
start_server 启动服务器
stop_server 停止服务器
server_list 查看所有服务器
search_server 搜索服务器
add_server 添加服务器
delete_server 删除服务器
backup_server 备份服务器
retracement 回档服务器
server_mods 打开所有服务器mod
server_start_batch_rewrite_run_memories 批量启动服务器并写入运行内存
build 创建服务器
rename_server 重命名服务器
redirected_server_path 修改服务器路径
latest_started_server 查看最新启动的服务器
banned 封禁服务器玩家或者ip
unban 解封服务器玩家或者ip
op 设置服务器玩家权限
deop 取消服务器玩家权限
help <命令> 查看帮助

# 程序类

命令 说明
help 查看帮助
version 查看版本
about 关于程序
sta 打开文件
output_program_info 输出程序信息
format_program 格式化程序
clear_cache 清除缓存
reset_settings 重置设置
exit 退出程序

# 修改类

命令 说明
change_server_properties 修改服务器配置文件
change_server_run_memories_config 修改运行内存配置文件
change_server_start_nogui 修改nogui状态
change_wait_server_eula_generate_time 修改等待服务器EULA生成时间
change_program_auto_startup 修改程序自动启动
change_get_minecraft_test_version 修改获取测试版本
change_storage_size_update_time 修改服务器存储大小更新时间
change_port 修改API端口
change_update_source 修改更新源

# 详细使用方法

# 服务器

  • 启动服务器
start_server <服务器名称>
  • 停止服务器
stop_server <服务器名称>
  • 查看所有服务器
server_list
  • 搜索服务器
search_server <服务器名称>
  • 添加服务器
add_server <服务器路径> <服务器名称> <服务器游戏版本> <核心类型>
  • 删除服务器
delete_server <服务器名称>
  • 回档服务器
retracement <服务器名称> <回档版本/不填默认None>
  • 打开服务器mod文件夹
server_mods <服务器名称>
  • 封禁服务器玩家或者ip
banned <玩家/ips> <服务器名称> <玩家名称/ip>
  • 解封服务器玩家或者ip
unban <玩家/ips> <服务器名称> <玩家名称/ip>
  • 设置服务器玩家权限
op <服务器名称> <玩家名称>
  • 取消服务器玩家权限
deop <服务器名称> <玩家名称>

# 程序

  • 查看版本
version
  • 关于程序
about
  • 打开文件
sta <文件路径>
  • 输出程序信息
output_program_info
  • 格式化程序
format_program
  • 清除缓存
clear_cache
  • 重置设置
reset_settings
  • 退出程序
exit

# 修改

  • 修改服务器配置文件
change_server_properties <服务器名称> <关键字> <值>
  • 修改运行内存配置文件
change_server_run_memories_config <最小内存> <最大内存>
  • 修改nogui状态
change_server_start_nogui <服务器名称> <true/false>
  • 修改等待服务器EULA生成时间
change_wait_server_eula_generate_time <服务器名称> <时间>
  • 修改程序自动启动
change_program_auto_startup <true/false>
  • 修改获取测试版本
change_get_minecraft_test_version <true/false>
  • 修改服务器存储大小更新时间
change_storage_size_update_time <服务器名称> <时间>
  • 修改API端口
change_port <端口号(范围:1024-65535)>
  • 修改更新源
change_update_source <更新源>
